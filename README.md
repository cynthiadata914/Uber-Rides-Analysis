# Project Title
## Uber-Rides-Analysis

## Goals
Build an interactive Excel dashboard that converts Uber ride data into operational and performance insights to improve revenue, service reliability, and driver management decisions.

## Objectives
This project aims to analyze Uber ride operations data to evaluate service performance, driver productivity, customer activity, and cancellation behavior. The dashboard is designed to support decision making for operational planning, driver incentives, and revenue optimization.
- Measure overall platform performance
- Evaluate driver productivity and service quality
- Understand customer engagement patterns
- Analyze ride demand by time of day
- Assess geographic ride distribution
- Examine cancellation trends and ride completion rates
- Provide actionable insights for business improvement

## Dataset description
### Source [Uber Ride dataset](https://docs.google.com/spreadsheets/d/1KgG0cSnDW_xrgKOS1M8s1AgnnrZKmvK3/edit?usp=drive_link&ouid=112873650373721064799&rtpof=true&sd=true)
The dataset contains ride level transactional data including request time, pickup/drop off locations, ride type, driver id, revenue, distance, ratings, and cancellation reasons.
- Request date/time
- Pickup/dropoff location
- Driver id
- Ride type
- Revenue
- Miles covered
- Rating
- Cancellation reason
- Customer/Driver feedback.

## Tools used
- Microsoft Excel
- Pivot tables
- Power query
- Dashboard design
- KPI calculations

## Data cleaning steps
- Removed duplicates
- Standardized date/time formats
- Extracted hour and time of day categories
- Handled missing values

## Anaalysis performed
This project involves a comprehensive analysis of uber ride data to evaluate operational performance, revenue generation, customer behavior, drivers performance, and service reliability.
The following analyses were performed using Excel pivot tables, calculated metrics, and interactive dashboard visuals:
- Ride volume and platform activity analysis.
- Revenue and financial performance analysis.
- Driver performance analysis.
- Customer engagement and satisfaction.
- Demand pattern and time based analysis.
- Location based analysis.
- Cancellation and service reliability analysis.
- Interactive filtering and drill down.

## Key insights
- Evening hours show the highest ride demand.
- Top 5 locations generate over 60% of total revenue.
- Driver 213 achieved the highest overall performance score.
- Long wait time and driver mismatch are leading to cancellation causes.
- High demand does not always translate to high revenue.

## Recommendations
- Focus marketing on high value locations.
- Increase drivers during peak hours.
- Reward top performing drivers.
- Introduce loyalty programs for frequent customers.
- Improve matching system to reduce cancellations.
